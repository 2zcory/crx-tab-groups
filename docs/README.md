# Project Docs

This folder contains the project-level documentation pack for `crx-tab-groups`.

The current pack is intentionally lean. It is designed to govern near-term delivery without pretending the project already needs a heavyweight enterprise document stack.

## Documents

- [PROJECT-DOCS.md](./PROJECT-DOCS.md): project classification, scope, actors, requirements, use cases, user flows, and current assumptions
- [ARCHITECTURE.md](./ARCHITECTURE.md): high-level and low-level design summary for the extension
- [DELIVERY-HANDOFF.md](./DELIVERY-HANDOFF.md): traceability from docs into execution slices and backlog
